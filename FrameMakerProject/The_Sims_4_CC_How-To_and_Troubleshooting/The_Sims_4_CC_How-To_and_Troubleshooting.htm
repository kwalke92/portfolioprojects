Installing Custom Content (CC) for The Sims 4 on Steam Deck


Introduction: This guide provides instructions on installing player-made custom content into The Sims 4, a popular life simulation video game on Steam Deck. The guide included a glossary, required items, prerequisites for custom content, finding the mods (short hand for modification) folder for The Sims 4, creating a shortcut for The Sims 4 and mods folder, downloading custom content, and troubleshooting common issues that may arise.


Glossary:

Custom Content- Player-made content for building (wall colors, floor styles, furniture) homes and create a sim (clothing, hairstyles, skin colors) for The Sims 4. Commonly abbreviated as ‘CC’.

Mods - Plural, short-hand for ‘modifications’ that can change the game, such as game play, mechanics, or add features.

Finding The Sims 4 Mods Folder

1.​From the main screen on the Steam Deck, select Power and select Desktop mode.

2.​Open Dolphin (the file explorer on Steam Deck).

3.​Go to the Home page.

4.​Click the hamburger menu and select “Show hidden” files. Select deck.

5.​Click on Local (these are normally hidden files)

6.​Click on Share

7.​Select Steam

8.​Select Steamapps

9.​Select compactdata

10.​Files are numbered instead of named. For the Sims 4, the number is 1222670.

11.​Select pfx

12.​Select drive_c

13.​Select users

14.​Select steamuser

15.​Select My Documents

16.​Select Electronic Arts

17.​Select The Sims 4

18.​Select the Mods folder

19.​(Optional) If you are using an SD card to store mods, 

-​Select Steamapps

-​Select common (games are placed here).

-​Mods still need to be added to the Mods folder on the Steam Deck.


Finished File Path


Create a Shortcut

Follow the file path detailed above (if you closed out of it). Select and drag the Sims 4 or the Mods folder to the left side menu bar in Dolphin.

Downloading CC

Account creation is necessary for some websites that have CC. The Sims Resource was used for this demo, and account creation is mandatory to get CC.


Select the CC you want after creating an account or logging in with a Google account.

Select download. You will see a screen that says your download is being prepared. While it is being prepared, the download button is grayed out. When ready, the download button becomes purple, and the screen says, “Your download is ready to go!” Select download.


Zip files are downloaded to the Steam Deck. Select the zip file you just downloaded, select extract in the upper left-hand of the screen


Troubleshooting


Issue: I don’t have a mods folder.

Solution: To have a mods folder, you must open the game at least once. A mods folder is automatically created.


Issue: My mods and CC are not showing up in-game.

Solution: There are three potential reasons why your mods and CC are not showing up in-game.

1.​Check if your mods and CC are installed in the correct mods folder.

2.​Make sure to extract the mods and CC from the zip file that you downloaded.

3.​Confirm that mods and CC are enabled in your game options.


Issue: The game updated, and my mods and CC are missing.

Solution: Updates can automatically disable mods and CC. When this happens, you must go into Game Options and re-enable them.


Issue: Do I need to extract every CC file? Can I leave them zipped?

Solution: You must extract every CC file and place them in the correct mods folder.


Issue: My loading screen CC has a flickering plumbob.

Solution: The plumbob flickering is a common issue. Unfortunately, you have to find a different loading screen that does not have the plumbob.
